# CMS Articles Service

Backend service for a CMS website to manage and display articles.

## Tech Stack

- NestJS
- TypeScript
- MongoDB (Mongoose)
- JWT Authorization

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Copy environment file:

```bash
cp .env.example .env
```

3. Start in development mode:

```bash
npm run start:dev
```

Base URL:

- `http://localhost:3000/api/v1`
